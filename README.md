# Larkhill Karate Club

Website rebuild for Larkhill Karate Club.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase Auth, Postgres, and Storage
- Vercel hosting

## Local Development

```bash
npm install
npm run dev
```

Create `.env.local` from `.env.example` when the Supabase project is ready.

## Current Focus

- Public website structure
- Accurate class timetable and contact information
- Staff gallery workflow for simple media uploads
- Accessibility-conscious staff screens with large controls and clear states
